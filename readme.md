# Расширение отладочной информации для СТЕКа

## ⚙️ Требования
- ОС: Windows 10 (только)
- .NET: Runtime 8.0 (обязательно)

## 🚀 Использование
1. Запустите `StackUTIL.exe`.
2. Запустите **СТЕК**.
3. Нажмите на запись в таблице и нажмите `Alt + Shift + F12`.
4. Скопируйте данные любым способом:
   - `Ctrl + C`
   - Кнопка **Копировать**
   - Клик по записи (поддерживается **мультиселект**)

## ⚠️ Важно
- Без `.NET 8` приложение не запустится.
- На Windows 11 работа не гарантируется.
